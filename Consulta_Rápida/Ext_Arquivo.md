# 📂 Extensões de Arquivo Suportadas

Lista das principais extensões e como o GitHub as renderiza no navegador.

### 1. Documentação e Texto
* **.md (Markdown):** Renderização nativa com formatação visual.
* **.txt:** Texto simples sem formatação.
* **.pdf:** Visualizador nativo integrado.
* **.adoc:** Documentação técnica formatada.

### 2. Programação e Scripts (Com Realce de Sintaxe)
* **.py:** Python.
* **.js / .ts:** JavaScript e TypeScript.
* **.dart:** Frameworks mobile e web.
* **.c / .cpp / .h:** Linguagens C e C++.
* **.ino:** Sketches para microcontroladores e Arduino.
* **.java:** Código-fonte Java.

### 3. Web e Design
* **.html:** Estrutura de páginas.
* **.css / .scss:** Estilização e design.
* **.svg:** Gráficos vetoriais (exibidos como imagem).

### 4. Dados e Configuração
* **.json:** Estruturas de dados e metadados.
* **.yaml / .yml:** Configurações de CI/CD e Docker.
* **.csv:** Tabelas (exibidas como grade interativa).
* **.xml:** Marcação de dados estruturados.

### 5. Acadêmico e Científico
* **.tex:** Código LaTeX (Renderiza fórmulas em arquivos .md).
* **.ipynb:** Jupyter Notebooks (Renderiza células e gráficos).

### Resumo de como fazer:

1.  **Imagens e GIFs:** A maneira mais simples é abrir o editor do seu repositório no GitHub e **arrastar o arquivo direto do seu computador** para dentro do texto. O GitHub gera o código automaticamente. Se quiser fazer manualmente, use: `![texto](link-da-imagem.gif)`.
2.  **Vídeos:** O processo é o mesmo (arrastar e soltar). O GitHub aceita arquivos MP4 e MOV. Ele criará um link que, ao salvar o arquivo, se transforma em um player de vídeo.
3.  **Dica para Organização:** Para manter o projeto limpo, é comum criar uma pasta chamada `assets` ou `images` dentro do seu repositório para guardar esses arquivos.
